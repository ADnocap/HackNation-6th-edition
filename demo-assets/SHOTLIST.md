## Fallbacks

- `DEMO_MODE` is the only mode — the hero path never touches the network. If a live call
  hangs, claims render `unverified` and flip as checks land: a slow check reads as the system
  working, not breaking.
- If a fixture site is down at recording time, the receipt's right pane renders the cached
  fetch (body + URL + timestamp) — say "cached fetch from 14:19 UTC" out loud instead of
  pretending it's live.
- Keep the first complete take, whatever its quality. It is the submission until a better one
  exists.
